# 🎯 Number Guessing Game (Python)

A simple and interactive number guessing game built using Python.  
The player has to guess a randomly generated number within limited attempts based on difficulty level.

---

## 🚀 Features

- 🎮 Multiple difficulty levels:
  - Easy (1–50, 5 attempts)
  - Medium (1–100, 7 attempts)
  - Hard (1–500, 9 attempts)
- 🔢 Random number generation
- 💡 Smart hints:
  - Too high / Too low
  - Close guesses
- 🔁 Replay option
- 🛡️ Input validation (handles invalid inputs)

---

## 🧠 How It Works

1. Choose a difficulty level
2. The computer selects a random number
3. Enter your guesses
4. Get hints:
   - "Too high" or "Too low"
   - "Close, but high/low"
5. Win by guessing correctly within limited attempts

---
